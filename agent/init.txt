  실행 순서

  # 1. Python Agent 서버 실행
  cd agent
  pip install -r requirements.txt
  cp .env.example .env  # ANTHROPIC_API_KEY 입력
  python scripts/init_rag.py  # 최초 1회
  uvicorn server:app --host 0.0.0.0 --port 8000 --reload

  # 2. Spring Boot 실행
  ./mvnw spring-boot:run

  # 3. React 실행
  cd front && npm start


  BIM AI 기능 예시 (채팅창에서)

  - "현재 프로젝트에 콘크리트 기둥 추가해줘" → BIM API 자동 호출
  - "5×3m 크기의 슬래브 만들어줘" → 크기 파싱 후 생성
  - "신규 프로젝트 만들어줘" → 프로젝트 생성
  - "현재 온도 알려줘" → DB 조회
  - "EMS 알림 있어?" → 알림 목록 조회




  라즈베리파이 실행 순서

  # 1. Ollama 서버 시작 (Docker 없이 직접)
  curl -fsSL https://ollama.ai/install.sh | sh
  ollama serve &

  # 2. 모델 다운로드 (~800MB)
  bash agent/scripts/pull_model.sh

  # 3. Docker Compose로 전체 실행
  docker-compose up -d

  또는 Docker Compose만으로:
  docker-compose up -d ollama
  docker exec twinspring-ollama-1 ollama pull llama3.2:1b-instruct-q4_K_M
  docker-compose up -d  # 나머지 서비스 시작
