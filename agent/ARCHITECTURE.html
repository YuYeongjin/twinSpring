2026 AI Agent 트렌드 평가
✅ ReAct Pattern — 업계 표준
Reason→Act→Observe 루프. 2024~2026 에이전트의 핵심 패턴.
✅ LLM Tool Calling — 현대적 접근
bind_tools() + ToolNode. 키워드 매칭 방식 완전 대체.
✅ LangGraph Workflow — 최신 프레임워크
StateGraph + 조건부 엣지. 2025 기준 멀티에이전트 표준.
✅ Persistent Checkpointing — 프로덕션급
PostgreSQL 영속 저장으로 대화 히스토리 세션 복원 가능.
✅ Supervisor Multi-Agent — 현대적 설계
라우터(1B) + 8개 도메인 서브에이전트 계층 분리.
✅ GraphRAG — 연구 최전선
Leiden 커뮤니티 감지로 단순 벡터 검색 대비 컨텍스트 품질↑.
✅ Subgraph 컴포지션 — 현대적 설계
build_react_subgraph() 팩토리로 도메인별 독립 서브그래프 조합.
✅ 경량 라우터 분리 — 효율적
llama3.2:1b를 분류 전용으로 분리해 응답 모델 호출 최소화.
⚠️ MCP 미구현 — Ollama 제약
Anthropic/Linux Foundation 표준. Ollama가 MCP 클라이언트 미지원.
Claude API 전환 시 적용 가능.
⚠️ A2A Protocol 미구현 — Google 표준
Agent-to-Agent 통신 표준. 현재 단일 서버 내 함수 호출로 대체 중.
⚠️ Parallel Tool Calls — 제한적
현재 단일 스레드 순차 실행. ToolNode는 병렬 지원하나 미활성화.
